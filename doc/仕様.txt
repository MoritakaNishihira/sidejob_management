＜機能概要＞

＜構成＞
side_job_management
 -- doc フォルダ
 -- src フォルダ
 -- db フォルダ
＜データベース＞
DBはSQliteを使う
DBファイルは db フォルダにある
テーブル
　JobName
　　id int
    JobName varchar(20)

　Side_Job_Management
　　JobDate date
　　JobStartTime time
　　JobEndTime time
    JobUpTime time
　　JobType int
　　Rewards money

＜画面構成＞
　・JsとHtmlとcssはsrcフォルダに格納する
　・cssは Material-UI のCSSを使う
　・cssのファイル名は Side_Job_Management.css
　ポータル画面（Portal.js）
　　・一覧画面を表示するエリアを設ける
　　・グラフエリアを表示するエリアを設ける
　　　・一覧画面のエリアとグラフエリアには 2mm の隙間を開ける

　一覧画面（List.js）
　　・一覧に表示する項目
　　　・稼働日付(Side_Job_Managementテーブルの JobDate)
　　　　・id JobDate
　　　　・書式 yyyy-MM-dd
　　　・稼働開始時間(Side_Job_Managementテーブルの JobStartTime)
　　　　・id JobStartTime
　　　　・書式 hh:mm
　　　・稼働終了時間(Side_Job_Managementテーブルの JobEndTime)
　　　　・id JobEndTime
　　　　・書式 hh:mm
　　　・実働時間(Side_Job_Managementテーブルの JobUpTime)
　　　　・id JobUpTime
　　　　・書式 hh:mm（60分単位）
　　　・業務名(jobnameテーブルの JobName、 KEYは JobName.id = Side_Job_Management.JobType)
　　　　・id JobName、
　　　・報酬(Side_Job_Managementテーブルの Rewards)
　　　　・id Rewards
　　　　・書式 #,##0
　　　・一覧の並び順は 稼働日付の降順

　　・年を入力するテキストボックス
　　　・数字のみ入力
　　　・初期値はシステム日付（年）

　　・表示ボタン
　　　・年の右隣に配置
　　　・クリックすると、入力された年をキーに side_job_management からデータを取得して一覧表示する
　　　　・年に 2025が入力された場合、2025/4/1～2026/3/31の範囲でデータを取得する
　　　　・年が未入力の場合、システム日付（年）で検索する
　　　　・データがない場合、「データが見つかりませんでした」のメッセージを表示する

　　・初期表示
　　　・年の初期値でside_job_management からデータを取得して一覧表示する

　　・一覧リスト
　　　・年の下に配置
　　　・稼働日をクリックすると詳細画面に遷移する

　詳細画面（Detail.js）

　グラフエリア
　　・グラフエリアに表示するグラフは以下の通り
　　・横には２個並べる（グラフが４つある場合は、縦横に２個ずつ並べる）
　　・グラフの縦横のサイズは、windowのサイズから案分する
　　・グラフ間には、2mmの隙間を開ける
　　

一覧画面から詳細画面に遷移する
　稼働日をクリックすると詳細画面に遷移する

一覧画面（list.js）
　詳細画面のすべての項目を表示する

詳細画面（detail.js）
　＜画面項目＞
　　稼働日
　　　年月日を入力
　　　カレンダーからも入力できる

　　稼働開始時刻(time)
　　　時刻を入力
　　　timePickerからも入力できる

　　稼働終了時刻(time)
　　　時刻を入力
　　　timePickerからも入力できる

　　稼働時間
　　　稼働開始時刻と稼働終了時刻から計算
　　　60分単位で計算する
　　　
　　業務名
　　　ドロップダウンから選択
　　　データソースは jobname テーブル

　　報酬
　　　手入力
　　　数値のみ
　　　桁区切りを入れる
